# 🌍 Banjara Translator

**बंजारा भाषा अनुवादक (Banjara Language Translator)**

> बंजारा (गोर बोली/पारसी/लंबाड़ी) भाषा को 100+ विश्व भाषाओं में अनुवाद करें और उल्टा भी! Translate Banjara language (also known as Gourboli, Parsee, Lambadi) to 100+ world languages and vice versa!

---

## 🚀 Features

- ✨ **Multi-language Support**: बंजारा ↔ 100+ विश्व भाषाएं
- 📚 **Grammar Guide**: Complete Banjara language grammar with examples
- 📖 **Dictionary**: Comprehensive Banjara-Hindi-English dictionary
- 🎨 **Beautiful UI**: Responsive, modern design with dark mode
- 🔄 **Powerful Translation Engine**: MyMemory API + Custom Dictionary
- ⌨️ **Keyboard Shortcuts**: Fast translation with Ctrl+Enter
- 📱 **Mobile Friendly**: Works perfectly on all devices
- 🌓 **Dark Mode**: Easy on the eyes, theme persistence
- 🔍 **Auto-detect**: Automatic language detection
- 💾 **Offline Ready**: Works with cached data


### 3️⃣ Choose Target Language

```
परिणाम: "पानी पीता हूँ"
```

## 🌍 About Banjara Language

**बंजारा भाषा** (Banjara Language), also known as:
- **गोर बोली** (Gourboli)
- **पारसी** (Parsee)
- **लंबाड़ी** (Lambadi)

### History
- Spoken by the Banjara community in India
- Indo-Aryan language family
- Mix of Hindi, Marathi, and regional influences
- Rich cultural heritage and oral traditions

### Distribution
- Primarily in: Maharashtra, Karnataka, Telangana, Andhra Pradesh
- Also found in: Madhya Pradesh and other Indian states
- Approximately 2-4 million speakers

### Cultural Significance
- Integral to Banjara identity
- Contains rich folklore and traditions
- Represents a unique cultural heritage
- Important for community cohesion
